## Run Locally

Clone the project

```bash
  git clone https://github.com/soydavidpaucar/react-typescript-todo-app.git
```

Go to the project directory

```bash
  cd react-typescript-todo-app
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn preview
```
